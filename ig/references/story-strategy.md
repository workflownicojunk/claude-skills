# Story Strategy: Types, Sequences, Retention

Comprehensive reference for Instagram Story production and optimization.

## Story Types

Five core story types. Rotate through all five weekly for variety.

| Type | Purpose | Frequency | Best Time |
|------|---------|-----------|-----------|
| Educational | Teach something specific in 3-5 slides | 2-3x/week | Morning (06:00-08:00) |
| Behind-the-scenes | Show real life, build authenticity | 2-3x/week | Midday or afternoon |
| Product / Promotion | Drive to offer, link sticker | 1-2x/week | Evening (18:00-20:00) |
| Poll / Question | Gather feedback, boost engagement | 1-2x/week | Anytime (high response all day) |
| Day-in-the-life | Full day documentation, lifestyle | 1x/week max | Throughout the day |

### Type 1: Educational Stories

**Format:** 3-5 slides, text-heavy, one concept per slide.

| Slide | Content | Example |
|-------|---------|---------|
| 1 (Hook) | Question or bold statement | "Weisst du, warum Protein-Timing egal ist?" |
| 2-3 (Body) | Explanation, one point per slide | "Dein Koerper speichert Aminosaeuren ueber Stunden..." |
| 4 (Takeaway) | Actionable summary | "Fazit: Iss dein Protein wann es dir passt. Die Gesamtmenge zaehlt." |
| 5 (CTA) | Save prompt or link | "Mehr dazu im neuen Reel. Link." |

### Type 2: Behind-the-Scenes

**Format:** 2-4 slides, casual, low production value is intentional.

Content ideas:
- Meal prep process
- Gym session (real, not staged)
- Working on new program content
- Reading/researching
- Team meeting or planning

**Rule:** BTS must feel genuinely candid. No staged "candid." No ring light selfies pretending to be caught off guard.

### Type 3: Product / Promotion

**Format:** 3-5 slides, structured pitch.

| Slide | Content |
|-------|---------|
| 1 | Pain point / problem |
| 2 | How the product solves it |
| 3 | Social proof (testimonial, result) |
| 4 | CTA + link sticker |

**Rule:** Max 2 promotional story sequences per week. More than that erodes trust and increases story skip rate.

### Type 4: Poll / Question

**Format:** 1-3 slides with interactive stickers.

| Sticker Type | Best Use | Example |
|-------------|----------|---------|
| Poll (2 options) | Binary choice, quick engagement | "Training morgens oder abends?" |
| Question sticker | Open-ended, content ideas, community building | "Was ist deine groesste Frage zum Thema Ernaehrung?" |
| Quiz | Educational, test knowledge | "Wie viel Protein pro kg Koerpergewicht? A) 0.8g B) 1.6g C) 2.5g" |
| Emoji slider | Emotional gauge, fun engagement | "Wie motiviert bist du heute?" |

**Rule:** Always respond to at least 5-10 question sticker replies via DM. This creates real conversations and signals to Instagram that the account generates meaningful interactions.

### Type 5: Day-in-the-Life

**Format:** 5-10 slides throughout the day. More slides acceptable here because they are spread over time.

**Structure:**
- Morning: routine, breakfast, mindset
- Midday: work, training, meals
- Evening: wind-down, reflection, teaser for tomorrow

**Rule:** Only 1x/week max. Overuse makes every day feel the same and reduces novelty.

## Slide Sequence Rules

### Optimal Slide Count

| Slide Count | Completion Rate | Recommendation |
|-------------|----------------|----------------|
| 1-3 slides | 85-95% | Quick tips, polls, single announcements |
| 4-5 slides | 65-80% | Educational sequences, promotions |
| 6-7 slides | 45-60% | Only for high-interest topics |
| 8-10 slides | 30-45% | Day-in-the-life only (spread over hours) |
| 10+ slides | <30% | Avoid. Split into 2 posting sessions. |

**Rule:** Default to 3-5 slides. Only exceed 5 for day-in-the-life content or extremely high-engagement topics.

### Slide Flow Architecture

| Slide Position | Function | Execution |
|---------------|----------|-----------|
| Slide 1 | Hook | Bold text, question, or striking visual. Must stop the auto-advance. |
| Slide 2-3 | Deliver | Core value, information, or entertainment |
| Slide 4 (second to last) | Curiosity bridge | Tease what's coming on the final slide |
| Slide 5 (last) | Payoff + CTA | The answer, the reveal, or the action item |

**Critical:** The transition from slide 1 to slide 2 has the highest drop-off. Slide 1 must create enough curiosity to compel the tap.

## Text Overlay Rules

| Rule | Details |
|------|---------|
| Font size | Large. If it's hard to read on a phone at arm's length, it's too small. |
| Contrast | Dark text on light backgrounds or light text on dark backgrounds. Never text on busy backgrounds without a semi-transparent overlay. |
| Reading time | Each slide must be readable in 2 seconds or less. That means max 15-20 words per slide. |
| Position | Center or upper-center of the frame. Avoid top 15% (username area) and bottom 20% (reply bar). |
| Animation | Minimal. Text that bounces or slides in adds time and annoys fast readers. Static text preferred. |
| Hierarchy | One main message per slide in large text. Optional supporting text in smaller font below. |

## Screen Safe Zones

```
+---------------------------+
|   TOP 15% - UI elements   |  <- Username, close button. No important content here.
|   (profile pic, name)     |
+---------------------------+
|                           |
|                           |
|   SAFE ZONE (65%)         |  <- All important text and interactive elements go here.
|   Place all key content   |
|   in this area.           |
|                           |
|                           |
+---------------------------+
|  BOTTOM 20% - UI elements |  <- Reply bar, send message. No text here.
|  (reply, message bar)     |
+---------------------------+
```

### Interactive Element Placement

| Element | Best Position | Why |
|---------|-------------|-----|
| Poll sticker | Center-lower (within safe zone) | Easy thumb reach for voting |
| Question sticker | Center | Needs keyboard, centered feels natural |
| Quiz sticker | Center-upper | Reading then selecting, eye starts at top |
| Emoji slider | Center | Natural left-to-right gesture area |
| Link sticker | Lower-center (above bottom safe zone) | "Swipe up" mental model persists |

## Retention Strategy

### Hook Techniques for Slide 1

| Technique | Example | Engagement Level |
|-----------|---------|-----------------|
| Bold claim | "Cardio ist NICHT der beste Weg zum Abnehmen." | Very high |
| Question | "Machst du diesen Fehler nach dem Training?" | High |
| Number + curiosity | "3 Lebensmittel, die ich JEDEN Tag esse." | High |
| Behind-the-scenes tease | "Das wollte ich euch schon lange zeigen..." | Medium-high |
| Poll without context | "Stimmt oder stimmt nicht?" (then reveal on slide 2) | Very high |

### Curiosity Bridges Between Slides

The goal: make each slide end with an incomplete thought.

| Bad (complete thought on one slide) | Good (bridge to next slide) |
|-----------------------------------|-----------------------------|
| "Protein ist wichtig fuer Muskelaufbau." | "Protein ist wichtig. Aber WANN du es isst..." (next slide: the answer) |
| "Ich trainiere 4x pro Woche." | "Ich trainiere 4x pro Woche. Aber Tag 3 ist anders..." |

## Story Highlights Organization

### Recommended Highlight Categories

| Highlight Name | Content | Icon Style |
|---------------|---------|-----------|
| Training | Workout tips, exercise demos, training plans | Dumbbell or muscle |
| Ernaehrung | Nutrition tips, recipes, meal prep | Plate or leaf |
| Ueber mich | Personal story, values, journey | Heart or photo |
| Ergebnisse | Client transformations, testimonials | Star or before/after |
| Produkte | BodyGuide, Coaching, program info | Shopping bag or link |
| FAQ | Common questions answered | Question mark |

**Rules:**
- Max 6-8 highlights (more = overwhelming, nobody clicks)
- Update highlights monthly (remove outdated stories, add fresh ones)
- First 4 highlights are most visible on mobile (prioritize these)
- Highlight covers should match brand aesthetic (cream/white/blue)

## Posting Frequency & Timing

| Guideline | Details |
|-----------|---------|
| Minimum | 3 stories/day for algorithm to consider you "active" |
| Optimal | 5-7 stories/day, mixed types |
| Maximum | 10-12 stories/day (only for launches or special events) |
| Consistency | Post stories every day. A day without stories reduces next-day reach. |
| Spacing | Spread stories throughout the day (not all at once) |

### Posting Schedule Template

| Time (CET) | Story Type | Notes |
|------------|-----------|-------|
| 06:00-07:00 | Morning routine / motivational | Catches early scrollers |
| 10:00-11:00 | Educational (1-3 slides) | Mid-morning break scrolling |
| 14:00-15:00 | BTS or poll | Afternoon engagement dip, interactive content helps |
| 18:00-19:00 | Product/promo or educational | Peak Instagram usage time |
| 21:00-22:00 | Casual / day recap | Evening wind-down scrollers |

## Story-to-Reel Pipeline

Stories are the testing ground for Reels. Use this workflow:

| Step | Action | Decision Criteria |
|------|--------|------------------|
| 1 | Post topic as a story (poll or educational) | Is this worth producing? |
| 2 | Measure: replies, poll votes, DM responses | >50 interactions = high interest |
| 3 | If high interest: produce a full Reel on this topic | Reel inherits a validated topic |
| 4 | Reference the story in the Reel caption | "Viele von euch haben in meiner Story gefragt..." |

**Rule:** Never produce a Reel on a topic that hasn't been validated in Stories first (exception: trending topics that require speed).

## Analytics: What to Track

| Metric | Where to Find | Benchmark (200K account) |
|--------|--------------|-------------------------|
| Story completion rate | Insights > Content > Stories | >70% for 3-slide, >50% for 5-slide |
| Reply rate | Count DM replies per story set | >0.5% of viewers |
| Sticker interaction rate | Insights > Interactions | >10% of viewers for polls |
| Link click rate | Insights > Link clicks | >2% of viewers |
| Exit rate per slide | Insights > Navigation | Slide 1 exit <20%, steady decline after |
| Reach vs. Followers | Insights > Reach | Stories should reach 10-20% of followers |

### Red Flags

| Signal | Meaning | Fix |
|--------|---------|-----|
| Slide 1 exit >30% | Hook is not compelling | Stronger opening statement or visual |
| Sudden drop on slide 3 of 5 | Middle slides are boring | Add curiosity bridge to slide 2 |
| Low poll participation (<5%) | Question is too generic or too niche | Ask binary, relatable questions |
| High "Next Story" taps | Content is predictable | Vary story types, add surprises |
| Declining reach week over week | Posting too infrequently or too much promo | Increase non-promotional stories |
