# Content Rules: Captions, CTAs, Hashtags

Definitive rules for caption writing, call-to-action hierarchy, and hashtag strategy for Instagram content.

## Caption Length & Algorithm Impact

| Caption Length | Algorithm Effect | When to Use |
|---------------|-----------------|-------------|
| <200 characters | Neutral to slight negative | Never. Too short to add value. |
| 200-500 characters | Neutral | Only for highly visual content where image tells the story |
| 500-1000 characters | Positive boost | Standard for most posts. Algorithm indexes caption text for topic matching. |
| 1000-2000 characters | Strong positive | Educational posts, transformation stories, deep-dive topics |
| 2000+ characters | Diminishing returns | Only if every sentence adds value. Padding hurts completion rate. |

**Rule:** Minimum 500 characters for every post. The 2026 algorithm uses caption text for semantic topic matching, which directly affects who sees the content.

## Caption Structure

Every caption follows this architecture:

```
[Hook Variation Opener - 1-2 lines]

[Value Body - 300-400 characters, the core insight]

[Line Break]

[CTA - one clear action]

.
.
.

[3-5 Hashtags]
```

### 1. Hook Variation Opener

The caption opener must be a DIFFERENT angle on the same topic as the visual hook. If the Reel's visual hook says "3 Fehler beim Krafttraining", the caption opener should NOT repeat that. Instead:

| Visual Hook | Bad Caption Opener | Good Caption Opener |
|-------------|-------------------|---------------------|
| "3 Fehler beim Krafttraining" | "Hier sind 3 Fehler beim Krafttraining" | "Ich sehe diese Fehler bei 9 von 10 Frauen im Gym." |
| "Warum du trotz Training nicht abnimmst" | "Viele Frauen fragen sich, warum sie nicht abnehmen" | "Dein Training ist nicht das Problem. Dein Alltag ist es." |
| "Hormon-Boost in 10 Minuten" | "Dieses 10-Minuten-Workout boosted deine Hormone" | "Cortisol runter, Testosteron rauf. Und du brauchst dafuer kein Gym." |

**Rule:** The caption opener expands, reframes, or emotionally anchors the visual hook. It never duplicates it. Think of it as the "why this matters to YOU" bridge.

### 2. Value Body

The core of the caption. 300-400 characters of actionable insight.

**Structure options:**
- **Problem-Solution:** Name the pain point, then give the answer
- **Myth-Bust:** State the common belief, then correct it with science
- **Story-Lesson:** Brief personal anecdote (2-3 sentences), then the takeaway
- **List format:** 3 specific tips (use line breaks, not bullet emojis)

**Voice rules for captions (adapt to creator's tone from account-baseline.md):**
- Match the creator's documented tone and communication style
- Accessible language (explain mechanisms in terms the target audience understands)
- First person when sharing experience, second person when teaching
- Short sentences for emphasis. Longer sentences for explanation.
- Vary rhythm. Not every sentence should be the same length.

### 3. Line Break

Always insert a visual break before the CTA. Use the dot-on-separate-lines method:

```
...letzte Zeile des Value Body.

.
.
.

Speicher dir diesen Post fuer spaeter.
```

This creates visual separation and makes the CTA feel intentional, not tacked on.

### 4. CTA (Call to Action)

One CTA per post. Never stack multiple CTAs.

## CTA Hierarchy

Ranked by algorithm value and conversion impact:

| Priority | CTA Type | Algorithm Value | German Example |
|----------|----------|----------------|----------------|
| 1 | Save | Highest | "Speicher dir diesen Post, damit du ihn beim naechsten Training parat hast." |
| 2 | DM-Send | Very high | "Schick diesen Post einer Freundin, die das hoeren muss." |
| 3 | Comment (specific) | High | "Schreib GUIDE in die Kommentare und ich schick dir den kostenlosen Plan." |
| 4 | Comment (open) | Medium | "Was ist dein groesster Struggle beim Training? Schreib's in die Kommentare." |
| 5 | Follow | Low | "Folge mir fuer mehr evidenzbasierte Trainingstipps." |

### CTA Selection Rules

| Content Type | Best CTA | Why |
|-------------|----------|-----|
| Educational / How-to | Save | User wants to reference later |
| Motivational / Mindset | DM-Send | User wants to share with someone who needs it |
| Product / Launch | Comment keyword (ManyChat) | Triggers automation funnel |
| Transformation / Results | Follow | New viewers who see results want more |
| Controversial / Opinion | Comment (open) | Leverage the debate instinct constructively |

### ManyChat Keyword Integration

When using comment-trigger CTAs, the keyword must be:
- ALL CAPS for visibility: "Schreib GUIDE in die Kommentare"
- Single word, easy to type: GUIDE, PLAN, REZEPT, START, GRATIS
- Unique per campaign (don't reuse keywords across active campaigns)
- Mentioned in both caption AND visual/audio for maximum trigger rate

## Hashtag Strategy

### Selection Process

1. **Start with topic:** What is this post about? (e.g., Krafttraining fuer Frauen in den Wechseljahren)
2. **Find niche tags:** Search Instagram for tags with 10K-500K posts (sweet spot)
3. **Check relevance:** Are the top posts under this hashtag similar to your content?
4. **Select 3-5:** Mix of topic-specific and audience-specific

### Hashtag Tiers (Example Structure)

| Tier | Range | Example Tags | Use Per Post |
|------|-------|-------------|-------------|
| Niche (target) | 10K-100K posts | #krafttrainingfrauen, #wechseljahrefitness, #hormonbalance | 2-3 |
| Mid-range | 100K-500K posts | #fitnessueber40, #frauengesundheit, #trainingstipps | 1-2 |
| Broad (avoid) | 500K+ posts | #fitness, #workout, #gesundheit | 0 |

### Forbidden Hashtags

Never use these. They are either too generic (zero distribution value) or flagged as spam-adjacent:

- `#fitness` `#fit` `#workout` `#gym` `#healthy` `#motivation`
- `#fitfam` `#fitspo` `#bodygoals` `#gains`
- `#followme` `#like4like` `#follow4follow`
- Any hashtag with 1M+ posts

### Hashtag Placement

Always at the very end of the caption, separated by the dot-line-break from the CTA. Never inline with caption text. Never in comments (2026 algorithm ignores comment hashtags).

## Forbidden Patterns

| Pattern | Why It's Forbidden | Alternative |
|---------|-------------------|-------------|
| Em dashes (the long dash) | #1 AI-generated text signal | Use comma, period, colon, or restructure |
| "In diesem Post zeige ich dir..." | Generic, zero hook value | Start with the insight or a question |
| Wall of text (no line breaks) | Unreadable on mobile, kills engagement | Break every 2-3 sentences |
| Emoji bullet points | Looks like a template, not authentic | Use line breaks and natural sentence flow |
| "Link in Bio!" as the only CTA | Wastes the CTA slot on a low-value action | Use Save/Send/Comment CTA, mention link in bio as secondary |
| Starting with "Wusstest du, dass..." | Overused opener, signals generic content | Reframe: state the fact as a provocative claim |
| Hashtags in first comment | Algorithm no longer indexes comment hashtags (2026) | Place hashtags at end of caption |

## Emoji Usage

| Rule | Example |
|------|---------|
| Sparingly, as emotional punctuation | "Das hat mich 3 Jahre gekostet. Aber es war jede Sekunde wert." (no emoji needed) |
| End of sentences, not beginning | "Dein Koerper kann mehr als du denkst. 💪" |
| Never as bullet points | Not: "💪 Tipp 1... 🔥 Tipp 2..." |
| Max 2-3 per caption | Overuse signals inauthenticity |
| Match creator's preferred emojis | Define in account-baseline.md (e.g., 💪 🔥 🤍 🌿 ✨) |
